# Western University – Share Your Story (Landing Page)

![Share Your Story preview](images/share-your-story-hero-1200w.jpg)

A responsive, accessible landing page created for the Western University Web Developer interview assignment. The page reflects Western's branding, encourages alumni engagement, and demonstrates semantic HTML, usability, and WCAG-aligned accessibility practices.

## Features

- Responsive layout with mobile-first design
- Western branding (colours, typography, tone)
- Accessible form with labels for First Name, Last Name, and Story
- Semantic structure (header, main, nav, section, footer)
- Keyboard-friendly navigation and visible focus states
- High-contrast, WCAG-compliant colour usage

## Technologies

- HTML5
- CSS (SASS)
- Vanilla JavaScript
